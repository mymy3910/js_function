JavaScript配列関数

JavaScript配列関数の動きを試せます。配列はカンマ（,）区切りで入力してください。

配列内要素の文字列と数値の区別はしていないのでそのまま入力してください。

配列

引数

結合したい配列を引数に記入してください。

concat

2つ以上の配列を結合して、新しい配列を返却。 引数に結合したい配列を指定する（※省略可）。 

            
            const array1 = ['a', 'b', 'c'];
            const array2 = [1, 2, 3];
            const array3 = array1.concat(array2);
            
            console.log(array3);
            // Expected output: Array ['a', 'b', 'c', 1, 2, 3]

every

配列内のすべての要素が、引数に指定した関数の処理に合格するかをテストし、true or falseを返却。 各要素に一度ずつテストし、合わない要素がある場合すぐにfalseを返す。 

            
            const array = [1, 2, 5, 9, 10, 15];
           
            console.log(array.every((x) => x > 20);
            // Expected output: true
            console.log(array.every((x) => x > 10);
            // Expected output: false

indexOf

配列の要素に、引数に指定された値の検索を実施。 値が見つからない場合は「-1」を返却。 見つかった場合は、検索した値が最初に見つかったインデックスを返却する。 

            
            const array1 = [1, 2, 3, 4];
            const array2 = ['dog', 'cat', 'bird'];

            console.log(array1.indexOf(2));
            // Expected output: 1

            console.log(array2.indexOf('rabbit'));
            // Expected output: -1

join

引数を配列の全要素に対して順に連結し文字列で返却。 引数は省略可能で、省略すると配列はカンマで区切られる。 空文字を渡すと、すべての要素が区切り文字なしで繋がれる。 

            
            const array = [1, 2, 3];
            
            console.log(array.join(/));
            // Expected output: "1/2/3"

map

引数として与えられた関数を配列のすべての要素に対して実施し、その結果を新しい配列して返却。 

            
            const array = [1, 2, 3, 10];
 
            console.log(array.map(x => x * 3));
            // Expected output: Array [3, 6, 9, 30]

slice

配列から引数で指定した範囲で取り出し、新しい配列を返却。 引数は取り出しの開始位置、終了位置を指定する（※省略可）。 省略すると配列の位置0から最期までを取り出す。 

            
            const array = ['a', 'b', 'c', 'd', 'e', 'f'];

            console.log(array.slice(1,3));
            // Expected output: Array ['b', 'c']
